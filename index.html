HTML Page Structure

!DOCTYPE

Tells the browser what HTML was used.

html

All code sits inside the html opening and closing tag

head

The head tag holds the title of the page and other meta data pertaining to the website, which is not visible on the browser

body

The body opening and closing tag houses all the html code

Inline vs Block Level Elements

Inline Elements

  • Do not start on a new line
  • Take only the necessary width

    • EXAMPLES: span, img, a tags

Block Elements

  • Start on a new line
  • Take the full width available

    • EXAMPLES: div, h1 - h6, p tag and form
