Creating an HTML file

  • Does NOT need a server
  • Files must end with the .html extension
  • Runs in a web browser (Chrome, Firefox, etc)
  • index.html is the root / home page of a website.

This is a Paragraph

  • Element names surroundes by angle brackets

  • End tag is usually the same but with a forward slash
  • Some tags close themselves (Remnant of XHTML)

HTML Page Structure

!DOCTYPE

Tells the browser what HTML was used.

html

All code sits inside the html opening and closing tag

head

The head tag holds the title of the page and other meta data pertaining to the website, which is not visible on the browser

body

The body opening and closing tag houses all the html code
